# Вариант 04 — Матрица прав (операция × роль)

Операции: Просмотр привычек, Создать/Редактировать/Удалить Habit, Просмотр отметок, Создать/Редактировать Entry, Управление напоминаниями, Управление целями, Управление пользователями, Просмотр статистики.

Роли: Администратор, Пользователь

Таблица (кто может что)

- Просмотр своих привычек: Пользователь
- Просмотр публичных привычек других: Пользователь
- Создать Habit: Пользователь
- Редактировать/Удалить Habit: Владелец, Администратор
- Создать/Редактировать Entry: Владелец привычки
- Удалить Entry: Владелец привычки, Администратор
- Управление напоминаниями (Reminder): Владелец привычки
- Управление целями (Goal): Владелец привычки
- Просмотр статистики (своей): Пользователь
- Просмотр статистики (чужой, если публичная): Пользователь
- Управление пользователями: Администратор
- Просмотр системного журнала: Администратор

Примечания:

- По умолчанию пользователь видит только свои привычки и публичные привычки других.
- Записи (Entry) привязаны к привычке — доступ имеют только владелец привычки и администратор.
- Администратор может редактировать/удалять любые сущности.
