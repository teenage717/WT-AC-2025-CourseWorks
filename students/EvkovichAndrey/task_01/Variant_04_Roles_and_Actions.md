# Вариант 04 — Хэбит‑трекер «Я молодец!» — Роли и действия

Краткое описание

MVP: Привычки, цели, отметки выполнения, напоминания, streaks, календарь, статистика. UI отображает календарь выполнения, графики прогресса и список текущих серий; API: /habits, /goals, /entries, /reminders, /stats. Приёмка (MVP): календарь выполнения, статистика, корректный расчёт streaks.

Роли и основные действия

- Администратор
  - CRUD для пользователей
  - Просмотр всех привычек и отметок
  - Модерация публичного контента
  - Доступ к системной статистике и журналам

- Пользователь
  - Создание, редактирование, удаление своих привычек
  - Настройка целей (Goal) для привычек
  - Добавление отметок выполнения (Entry) за разные дни
  - Настройка напоминаний (Reminder)
  - Просмотр календаря выполнения, графиков прогресса
  - Просмотр текущих и рекордных серий (streaks)
  - Просмотр публичных привычек других пользователей (опционально)

- Система (фоновые задачи)
  - Расчёт и обновление streaks
  - Отправка напоминаний (push/email)
  - Генерация сводной статистики

Примеры сценариев

- Пользователь создаёт привычку «Читать 30 минут ежедневно», ставит цель «30 дней подряд», добавляет отметки выполнения → система считает streaks и показывает прогресс в календаре.
- Пользователь получает напоминание в указанное время, отмечает выполнение → streak обновляется, если выполнение последовательное.
- Администратор видит общую статистику по активности пользователей, может деактивировать неактивных.
