# Вариант 04 — Хэбит‑трекер «Я молодец!» — Маршруты (user flows)

1. Идентификация (вход в систему)

   - Гость: регистрация / логин → личный кабинет
   - Пользователь: логин → дашборд с календарём и текущими привычками

2. Создание привычки (Пользователь)

   - Заполнение названия, описания, периодичности
   - Опционально: установка цели (Goal)
   - Опционально: настройка напоминаний (Reminder)

3. Отметка выполнения (Пользователь)

   - Выбор привычки и даты
   - Ввод значения выполнения (например, 1 — выполнено, 0.5 — частично)
   - Добавление комментария (опционально)
   - Система автоматически пересчитывает streak

4. Просмотр прогресса (Пользователь)

   - Календарь выполнения (дни с отметками выделены)
   - Графики прогресса по привычке (линия/столбцы)
   - Текущие и рекордные streaks

5. Управление напоминаниями (Пользователь)

   - Список активных напоминаний
   - Добавление/редактирование/удаление напоминания

6. Администрирование (Администратор)

   - Просмотр списка пользователей
   - Просмотр всех привычек и отметок
   - Модерация публичного контента
   - Доступ к системным журналам

7. Отчёты и статистика

   - Сводка за месяц/год
   - Экспорт данных (CSV/PDF)
   - Публичный профиль (если привычки публичные)
