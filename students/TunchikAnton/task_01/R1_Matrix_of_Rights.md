| Операция | Admin | User | Guest |
| --- | :---: | :---: | :---: |
| Регистрация | ✓ | — | ✓ |
| Вход (получить JWT) | ✓ | ✓ | ✓ |
| Просмотр списка задач | ✓ | ✓ (свои) | ✗ |
| Создать задачу | ✓ | ✓ (свои) | ✗ |
| Редактировать задачу | ✓ | ✓ (свои) | ✗ |
| Удалить задачу | ✓ | ✓ (свои) | ✗ |
| Создать задачу с подзадачами | ✓ | ✓ | ✗ |
| Управление подзадачами (create/toggle/delete) | ✓ | ✓ (в своих задачах) | ✗ |
| Просмотр тегов | ✓ | ✓ | ✗ |
| Создать тег | ✓ | ✓ | ✗ |
| Создать новые теги при создании задачи | ✓ | ✓ | ✗ |
| Загрузка/просмотр/скачивание/удаление файлов | ✓ | ✓ (в своих задачах) | ✗ |
| Создание/просмотр/удаление напоминаний | ✓ | ✓ (в своих задачах) | ✗ |
| Календарь (due_at) | ✓ | ✓ (свои) | ✗ |
| Повтор: generate-next | ✓ | ✓ (свои) | ✗ |
| NLU (заглушка) | ✓ | ✓ | ✗ |
| Управление пользователями (список/роль) | ✓ | ✗ | ✗ |

**Примечания:**
- User ограничен scope: доступ только к собственным сущностям.
- Admin имеет доступ ко всем сущностям и к админ-операциям.
